## What It Is

weballoon is a desktop workspace browser that turns websites into clean, app-like experiences.
It gives each app its own isolated session, helps you organize work by workspace, and keeps your
data local by default.

Instead of piling every service into one browser profile, weballoon lets you separate tools by
client, project, or workflow without losing convenience.

## Why People Use It

- Turn any website into a dedicated desktop app
- Keep cookies and sessions isolated between apps
- Group apps by workspace for better focus
- Store browsing data locally on the device
- Sync app and workspace structure to the cloud without sending browsing data or logins
- Check plans and billing from the linked device flow
- Receive production-ready desktop updates through packaged releases

## Highlights

### Isolated Sessions

Each app runs in its own persistent partition, so logins, cookies, and storage do not bleed into
other apps.

### Workspace Organization

Create focused setups for different contexts like sales, design, operations, or personal use.

### Privacy-First by Default

weballoon is built around a simple promise: your browsing data stays on your machine unless you
explicitly choose otherwise. Optional cloud sync is limited to app and workspace structure, never
browsing content, account sessions, or login state.

### Cloud Sync Without Browsing Data

When enabled, sync carries your app catalog and workspace layout between devices. Credentials,
cookies, browsing history, and local web storage remain local.

### Native Desktop Flow

The app supports deep linking, desktop packaging, account linking, diagnostics, and packaged
release updates for supported builds.

## Installation

### macOS

1. Download the `.dmg` from the latest release.
2. Move `weballoon` to `Applications`.
3. Open it from `Applications`.
4. If macOS warns on first launch, use right-click → `Open`.

### Windows

1. Download the installer from the latest release.
2. Run the installer.
3. Launch weballoon from the Start menu.

### Linux

1. Download the `.AppImage`.
2. Mark it executable with `chmod +x`.
3. Run the file.

## Privacy Promise

weballoon is designed to respect the boundary between productivity and surveillance.

- No browsing data is sent during normal app use
- No shared cookie jar across isolated apps
- No forced cloud sync for local workflows
- Optional diagnostics can be disabled
- Diagnostic reports are sanitized and intended for stability work, not behavioral tracking
